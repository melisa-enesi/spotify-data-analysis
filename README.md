# netflix-data-analysis

Exploratory data analysis of Netflix tracks using Python, Pandas, and visualization.



\# Netflix Data Analysis and Recommendation System



This project explores trends in Netflix’s catalog using data analysis, visualization, and a simple machine learning recommendation system.



The goal of this project was to practice an end-to-end data workflow: data loading, cleaning, exploration, visualization, modeling, and communicating insights.



---



\## Dataset



The dataset contains information about Netflix movies and TV shows, including:

\- Title

\- Type (Movie / TV Show)

\- Country

\- Release year

\- Rating

\- Duration

\- Genre



---



\## Exploratory Data Analysis



The following questions were explored:



\- How has Netflix content grown over time?

\- What percentage of Netflix content is Movies vs TV Shows?

\- Which countries produce the most Netflix content?

\- What ratings dominate the platform?

\- What is the typical movie duration?

\- What genres are most common?

\- How have Movies vs TV Shows changed over time?



---



\## Key Insights



Content Growth  

Netflix content production increased dramatically after 2015, with rapid growth between 2016–2019 and a slight decline after 2019.



Content Type  

Movies make up the majority of Netflix’s catalog, although TV Shows have grown significantly in recent years.



Top Producing Countries  

The United States produces the most Netflix content by a large margin, followed by India and the United Kingdom.



Ratings  

TV-MA and TV-14 dominate the platform, indicating a strong focus on mature audiences.



Movie Duration  

Most movies fall between 80 and 120 minutes.



Top Genres  

The most common genres include International Movies, Dramas, Comedies, International TV Shows, and Documentaries.



---



\## Recommendation System



A simple content-based recommender system was built using TF-IDF vectorization and cosine similarity.  

The model recommends similar titles based on genre and rating.



Example:

recommend("Sankofa")
